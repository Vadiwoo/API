MAKING API CALLS WITH JAVASCRIPT

HTTP Request Format

In javascript, we make HTTP request through AJAX. AJAX stands for Asynchronous Javascript and XML. It is the use of XMLHttpRequest object to commnicate with server-side scripts. We can send and recieve information in a variety of formats, including JSON, XML, and also text files. In this tutorial we are going to recieve information in JSON format by using GET method.

Here is the javascript application that use in this tutorial

1.In this application we creates a new XMLHttpRequest by with line:-

var request = new XMLHttpRequest();

2.In the next line, we specify the request type:-

request.open(requestMethod, queryURL + queryString, async);

3.In the next line, we specify what do we do with the responses we recieve from the request.

request.addEventListener("load", function(){...});

3.Finally we send our request.

request.send(JSON.stringify(payload));
